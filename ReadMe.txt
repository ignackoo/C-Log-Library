TRACE=1, DEBUG=2, INFO=4, WARNING=8, ERROR=16, CRITICAL=32,

message log format

date       time     code     type     message
YYYY-MM-DD HH:MM:SS XXXXXXXX    TRACE message to log.\n\r

YYYY-MM-DD HH:MM:SS XXXXXXXX     INFO message to log.\n\r

YYYY-MM-DD HH:MM:SS XXXXXXXX    ERROR message to log.\n\r
